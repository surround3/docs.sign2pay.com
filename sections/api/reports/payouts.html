Payouts

The Payouts API gives you access you your latest and still pending payouts. Optionally, you can limit your results by supplying a specific date.

This API provides access for both create and update client applications.

Note: These are Token Authorized calls.

Payout by Date

GET https://app.sign2pay.com/api/v2/reports/payouts/date/2015-04-14

Passing a date param will return payouts generated on that date, or an empty array if none exist.

Payout by Type

GET https://app.sign2pay.com/api/v2/reports/payouts/pending
Parameter Required Type Description
type no String Payout type you are looking for "latest" | "pending"

Request Headers & Body

Accept: */*
Accept-Encoding: gzip, deflate
Content-Type: application/json
Authorization: Token token=2d7c34be6b03de66386e50a962dd4349
Accept-Language: en-us
