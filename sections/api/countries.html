Countries

The Countries API provides you with an arrary of countries that Sign2Pay is currently processing payments in. This means that your user must have a bank account from one of these countries.

List Supported Countries

GET https://app.sign2pay.com/api/v2/countries

Request Headers & Body

Accept: */*
Accept-Encoding: gzip, deflate
Content-Type: application/json
Accept-Language: en-us
