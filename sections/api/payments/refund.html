Payments: Refund

This API allows you to refund a payment given a specific id and amount.

Note: This is a Token Authorized call.

Refund a Payment

POST https://app.sign2pay.com/api/v2/purchases/55fac963e6db9909ba000003/refunds
Parameter Required Type Description
amount yes Integer Amount to refund in cents.

Request Headers & Body

Accept: */*
Accept-Encoding: gzip, deflate
Content-Type: application/json
Authorization: Token token=2d7c34be6b03de66386e50a962dd4349
Accept-Language: en-us

{"refund": {"amount": 500}}
